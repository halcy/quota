Demo for quota.pl

Parameters

  • type - Filetype for the output (svg, png). Default: svg.
  • phrase - Search phrase for the trends. Default: Google.
  • symbol - Stock symbol for the stock graph. Default: GOOG.
  • w - Width for the resulting image. Default: 900.
  • h - height for the resulting image. Default: 400.
  • style - Stylesheet URL for custom graph styling. Only works in svg mode. Have a look at the documentation for SVG::TT::Graph if you are interested in using this. Default: graph.css
  • from - Start date for the graph. Default: One year in the past.
  • to - End date for the graph. Default: Now.
